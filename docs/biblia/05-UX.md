# UX

## Navegación

- Logo y barra de módulos siempre visibles.
- Un único área de trabajo.
- No existe navegación por páginas.
- El módulo activo se conecta al área de trabajo mediante un pulso visual.
- El módulo anterior se desconecta simultáneamente.

## Notificaciones

- Toast discretos.
- No bloqueantes.
- No invasivos.

## Búsqueda

Búsqueda global basada en relaciones.

Una búsqueda debe devolver toda la información asociada a la entidad buscada.

## Modales

Solo para tareas puntuales (altas, confirmaciones, selección de archivos, etc.).
