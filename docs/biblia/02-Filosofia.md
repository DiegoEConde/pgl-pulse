# Filosofía

## Juramento de Pulse

Diseñamos Pulse para que la tecnología desaparezca detrás del trabajo.

Cada función debe aportar claridad.
Cada módulo debe simplificar una tarea.
Cada animación debe tener un propósito.
Cada dato debe existir una sola vez.

La Unidad será siempre el centro del sistema.
La simplicidad tendrá prioridad sobre la complejidad.

Pulse no busca impresionar.
Pulse busca transmitir seguridad, tranquilidad y control.

Porque cuando Pulse late, PGL funciona.

## Principios

1. La Unidad es el centro del universo de Pulse.
2. Pulse nunca olvida.
3. Pulse nunca interrumpe sin una razón crítica.
4. Pulse informa. El usuario decide.
5. Pulse observa, recuerda y sugiere.
6. La tecnología cambia. La filosofía permanece.
