# Design System (PPDS)

## Identidad

- Modo oscuro.
- Azul + blanco.
- Estética tecnológica.
- Inspiración en el concepto de 'Pulse'.

## Movimiento

Toda la energía desciende desde el módulo activo hacia el área de trabajo.

Las animaciones representan acciones reales, nunca decoración.

## Lenguaje visual

- Azul: información y energía.
- Verde: correcto.
- Amarillo: atención.
- Rojo: integridad comprometida.
